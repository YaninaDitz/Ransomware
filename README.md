# Ransomware
This repository contains Python implementations for detecting ransomware activities in Bitcoin blockchain transactions using advanced machine learning techniques. The project includes comprehensive data analysis, preprocessing pipelines, and multiple predictive models.

## Features
- Exploratory Data Analysis (EDA)

## Comprehensive statistical analysis of Bitcoin transaction patterns
- Visualization of blockchain data distributions and anomalies
- Feature correlation analysis and temporal trend identification

## Data Preprocessing
- Bitcoin blockchain data cleaning and normalization
- Feature engineering for transaction pattern recognition
- Data scaling and preparation for machine learning models

## Machine Learning Models
- Logistic Regression (LR): Baseline classification model for ransomware detection
- XGBoost: Gradient boosting implementation for enhanced pattern recognition
- LSTM (Long Short-Term Memory): Deep learning approach for sequential transaction analysis

## Technologies
- Pandas, NumPy for data manipulation
- Scikit-learn for traditional ML models
- XGBoost for gradient boosting
- TensorFlow/Keras for LSTM implementation
- Matplotlib/Seaborn for data visualization
